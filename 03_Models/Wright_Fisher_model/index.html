    1. What the Wright–Fisher model is¶

    The Wright–Fisher model is a discrete-time stochastic population model that describes how allele frequencies change across generations due to random sampling of gametes in a finite population.

    It provides the canonical mathematical foundation for genetic drift.

    Genetic drift is the phenomenon;
    the Wright–Fisher model is the formal mechanism.

    2. Core assumptions¶

    The classical Wright–Fisher model assumes:

    1. Finite population size

      • Diploid population of constant size ( \(N\) )

      • Total number of gene copies: ( \(2N\) )

    2. Non-overlapping generations

      • Entire population is replaced each generation

    3. Random mating

      • No population structure

      • No assortative mating

    4. No selection, mutation, or migration

      • All alleles are selectively neutral

      • Allele frequency changes are purely stochastic

    5. Sampling with replacement

      • Each new gene copy is drawn independently from the parental generation

    These assumptions define a minimal neutral model.

    3. State variable¶

    Let:

    • ( \(p_t\) ): frequency of allele ( \(A\) ) at generation ( \(t\) )

    The state space is:

    \[ p_t \in \{ 0, \frac{1}{2N}, \frac{2}{2N}, \dots, 1 \} \]

    This discreteness is fundamental.

    4. Transition rule (binomial sampling)¶

    Given allele frequency ( \(p_t\) ), the number of ( \(A\) ) alleles in the next generation follows:

    \[ X_{t+1} \sim \mathrm{Binomial}(2N, p_t) \]

    and therefore:

    \[ p_{t+1} = \frac{X_{t+1}}{2N} \]

    This single equation fully defines the Wright–Fisher process.

    5. Expectation and variance¶

    Expectation¶

    \[ \mathbb{E}[p_{t+1} \mid p_t] = p_t \]

    ➡️ Drift does not bias allele frequency on average.

    Variance¶

    \[ \mathrm{Var}(p_{t+1} \mid p_t) = \frac{p_t(1 - p_t)}{2N} \]

    Key consequences:

    • Drift strength increases when:

      • \(N\) is small

      • \(p_t\) is intermediate

    • Variance goes to zero at \(p = 0\) and \(p = 1\)

    6. Absorbing boundaries and fixation¶

    • \(p = 0\) : loss of allele \(A\)

    • \(p = 1\): fixation of allele \(A\)

    These are absorbing states:

    \[ p_t = 0 \Rightarrow p_{t+k} = 0 \quad \forall k > 0 \]

    Under pure drift:

    • Fixation is inevitable

    • Which allele fixes is random

    7. Fixation probability¶

    For a neutral allele:

    \[ \Pr(\text{fixation} \mid p_0) = p_0 \]

    In particular:

    • A new mutation appears at frequency ( \(\frac{1}{2N}\) )

    • Fixation probability:

    \[ \Pr(\text{fixation}) = \frac{1}{2N} \]

    This result is one of the most important in population genetics.

    8. Time to fixation (order of magnitude)¶

    • Expected time to fixation or loss:
      $$ \mathcal{O}(N)
      $$

    • Conditioning on fixation:
      $$ \mathcal{O}(4N)
      $$

    Drift is slow in large populations, fast in small ones.

    9. Relation to genetic drift¶

    Genetic drift is:

    The stochastic change in allele frequency caused by finite population sampling.

    The Wright–Fisher model shows that drift is:

    • Unbiased in expectation

    • Inevitable in finite populations

    • Strongly dependent on population size

    10. From Wright–Fisher to diffusion approximation¶

    Taking the limit:

    • \(N \to \infty\)

    • Small changes per generation

    leads to the diffusion approximation:

    \[ dp = \sqrt{\frac{p(1-p)}{2N}} \, dW_t \]

    This connects Wright–Fisher to:

    • Stochastic differential equations

    • Fokker–Planck equations

    • Continuous-time allele frequency dynamics

    11. Position in the dynamics framework¶

    Level Role
    Genetic drift Evolutionary phenomenon
    Wright–Fisher model Discrete stochastic model
    Diffusion approximation Continuous limit
    Coalescent theory Backward-time dual

    12. What the Wright–Fisher model does not include¶

    • Selection → added via biased sampling

    • Mutation → added via mutation matrices

    • Migration → added via structured populations

    • Linkage → single-locus only

    These extensions define new models, not modifications of the core.

    13. Conceptual summary¶

    The Wright–Fisher model is the simplest mathematically complete description of evolution under randomness alone.

    It is the probabilistic backbone underlying:

    • Genetic drift

    • Neutral theory

    • Diffusion theory

    • Coalescent theory

    • GWAS kinship modeling (via covariance structure)

    因为在 Wright–Fisher 模型里，给定上一代等位基因频率 (p_t)，下一代中该等位基因的拷贝数 (K_{t+1}) 被建模为一个二项分布：

    \[ K_{t+1}\mid p_t \sim \mathrm{Bin}(2N,\; p_t). \]

    这里的含义是：

    • 群体为二倍体、大小固定为 (N)，所以下一代总共有 (2N) 条等位基因拷贝；

    • 形成下一代时，相当于从上一代的等位基因库中 独立地抽取 (2N) 次（随机配子抽样）；

    • 每次抽到目标等位基因（比如 (A)）的概率就是当前频率 (p_t)。

    为什么 \(\mathbb{E}[K_{t+1}\mid p_t]=2N p_t\)？¶

    二项分布的一个基本性质：若 (X\sim \mathrm{Bin}(n,p))，则

    \[ \mathbb{E}[X]=np. \]

    你也可以用“指示变量”从头推一遍（更直观）：

    令
    $$ I_i=
    \begin{cases}
    1,\text{第 }i\text{ 次抽样抽到目标等位基因}\
    0,\text{否则}
    \end{cases}
    $$ 那么
    $$ K_{t+1}=\sum_{i=1}^{2N} I_i.
    $$

    在给定 \(p_t\)的条件下，每次抽到目标等位基因的概率是\(p_t\)，所以 $$ \mathbb{E}[I_i\mid p_t]=1\cdot p_t+0\cdot (1-p_t)=p_t.
    $$

    利用期望的线性性（不需要独立性也成立）： $$ \mathbb{E}[K_{t+1}\mid p_t]
    =\mathbb{E}!\left[\sum_{i=1}^{2N} I_i ,\middle|, p_t\right]
    =\sum_{i=1}^{2N} \mathbb{E}[I_i\mid p_t]
    =\sum_{i=1}^{2N} p_t
    =2N p_t.
    $$

    顺带一提：这一步马上推出 $\(\mathbb{E}[p_{t+1}\mid p_t]=p_t\)$¶

    因为 $\(p_{t+1}=K_{t+1}/(2N)\)$：

    \[ \mathbb{E}[p_{t+1}\mid p_t] =\mathbb{E}!\left[\frac{K_{t+1}}{2N}\middle|p_t\right] =\frac{1}{2N}\mathbb{E}[K_{t+1}\mid p_t] =\frac{1}{2N}\cdot 2N p_t =p_t. \]

    这句话的直觉是：纯漂变在平均意义上不改变等位基因频率（但方差不为 0，所以会随机波动并最终吸收在 0 或 1）。

    如果你愿意，我也可以紧接着用同样思路把
    (\mathrm{Var}(K_{t+1}\mid p_t)) 和 (\mathrm{Var}(p_{t+1}\mid p_t)) 完整推出来，并解释边界 (p=0,1) 的“吸收态”含义。

    好，接着把 (\mathrm{Var}(p_{t+1}\mid p_t)) 推出来，并把边界 (p=0,1) 讲清楚（这正是后面 diffusion approximation 的核心输入）。

    1) 仍从 Wright–Fisher 的抽样假设出发¶

    假设（Wright–Fisher drift 部分）

    • 二倍体、群体大小恒定 (N)，每代共有 (2N) 份等位基因拷贝。

    • 下一代的 (2N) 份拷贝来自上一代的随机配子抽样：每次抽到目标等位基因（记为 (A)）的概率是当前频率 (p_t)。

    • 因此
      [
      K_{t+1}\mid p_t \sim \mathrm{Bin}(2N,,p_t),
      \qquad
      p_{t+1}=\frac{K_{t+1}}{2N}.
      ]

    你上一条已经得到
    [
    \mathbb{E}[K_{t+1}\mid p_t]=2N p_t
    \quad\Rightarrow\quad
    \mathbb{E}[p_{t+1}\mid p_t]=p_t.
    ]

    2) 推导 (\mathrm{Var}(K_{t+1}\mid p_t))¶

    二项分布的性质：若 (X\sim \mathrm{Bin}(n,p))，则
    [
    \mathrm{Var}(X)=np(1-p).
    ]

    所以直接得：
    [
    \mathrm{Var}(K_{t+1}\mid p_t)=2N,p_t(1-p_t).
    ]

    如果你想“从头推”也可以用指示变量法（更扎实）：

    • 令 (I_i) 为第 (i) 次抽样是否抽到 (A)，则 (I_i\mid p_t\sim \mathrm{Bernoulli}(p_t))。

    • (K_{t+1}=\sum_{i=1}^{2N} I_i)。

    • (\mathrm{Var}(I_i\mid p_t)=p_t(1-p_t))。

    • 在 Wright–Fisher 的独立抽样假设下，(\mathrm{Cov}(I_i,I_j\mid p_t)=0)（(i\neq j)）。
      于是
      [
      \mathrm{Var}(K_{t+1}\mid p_t)
      =\sum_{i=1}^{2N}\mathrm{Var}(I_i\mid p_t)
      =2N,p_t(1-p_t).
      ]

    3) 推导 (\mathrm{Var}(p_{t+1}\mid p_t))¶

    因为 (p_{t+1}=K_{t+1}/(2N))，用方差的缩放性质 (\mathrm{Var}(aX)=a^2\mathrm{Var}(X))：

    [
    \mathrm{Var}(p_{t+1}\mid p_t)
    =\mathrm{Var}!\left(\frac{K_{t+1}}{2N}\middle|p_t\right)
    =\frac{1}{(2N)^2}\mathrm{Var}(K_{t+1}\mid p_t)
    =\frac{1}{(2N)^2}\cdot 2N p_t(1-p_t)
    =\frac{p_t(1-p_t)}{2N}.
    ]

    这就是漂变一步的“噪声强度”，后面做 diffusion approximation 时会变成扩散项系数。

    4) 边界 (p=0,1) 的含义（为什么重要）¶

    在 Wright–Fisher 里：

    • 若 (p_t=0)，表示 (A) 在群体里 已经没有拷贝。此时
      [
      K_{t+1}\mid p_t=0 \sim \mathrm{Bin}(2N,0)\Rightarrow K_{t+1}=0 \text{（必然）}
      ]
      所以 (p_{t+1}=0)。这叫 吸收态 (absorbing boundary)：一旦丢失就不会靠漂变“自己回来”。

    • 若 (p_t=1)，表示群体里 全是 (A)。此时
      [
      K_{t+1}\mid p_t=1 \sim \mathrm{Bin}(2N,1)\Rightarrow K_{t+1}=2N \text{（必然）}
      ]
      所以 (p_{t+1}=1)。同样是吸收态：一旦固定也不会靠漂变“自己掉下去”。

    并且注意：在边界处方差也为 0：
    [
    \mathrm{Var}(p_{t+1}\mid p_t)=\frac{p_t(1-p_t)}{2N}
    ]
    当 (p_t=0) 或 (1) 时，(,p_t(1-p_t)=0)，随机波动消失——完全符合“吸收”的直觉。

    5) 一句总结（把这步和 diffusion 接上）¶

    你现在已经有了 drift 的两件关键东西：

    [
    \mathbb{E}[p_{t+1}-p_t\mid p_t]=0,
    \qquad
    \mathrm{Var}(p_{t+1}-p_t\mid p_t)=\frac{p_t(1-p_t)}{2N}.
    ]

    后面做 diffusion approximation 时，第一条对应 漂移项（drift term）=0（纯中性漂变），第二条对应 扩散项（diffusion term）(\propto p(1-p))，并且边界 (0,1) 仍然是吸收边界（除非你加入突变/迁移等机制）。

    如果你愿意，我们下一步就顺势进入 diffusion：把离散时间的
    (\Delta p = p_{t+1}-p_t) 做尺度变换（比如时间按 (2N) 缩放），写出对应的 SDE / Fokker–Planck 方程，并解释为什么扩散系数是 (p(1-p))。你想走 SDE 路线 还是 Fokker–Planck 路线？（我可以按你选的那条直接推，不用你再补背景。）

    下面我把这两条 从 Wright–Fisher 的定义开始，一步不跳地推到你写的结果：

    [
    \mathbb{E}[p_{t+1}-p_t\mid p_t]=0,\qquad
    \mathrm{Var}(p_{t+1}-p_t\mid p_t)=\frac{p_t(1-p_t)}{2N}.
    ]

    0) 模型设定与符号¶

    Wright–Fisher（仅漂变，中性）假设：

    1. 二倍体群体，成体大小固定为 (N)，所以每代有 (2N) 份等位基因拷贝。

    2. 从第 (t) 代到第 (t+1) 代的形成过程：把第 (t) 代的基因库当作一个“无限混合池”，对下一代的 (2N) 份等位基因拷贝进行随机抽样。

    3. 给定第 (t) 代目标等位基因（记为 (A)）的频率 (p_t)，则每次抽到 (A) 的概率都是 (p_t)，并且（在 WF 的标准版本里）各次抽样独立。

    定义：

    • (K_{t+1})：第 (t+1) 代中 (A) 的拷贝数（在 (2N) 份里有多少份是 (A)）。

    • (p_{t+1} := \dfrac{K_{t+1}}{2N})。

    于是（关键一步）：

    [
    K_{t+1}\mid p_t \sim \mathrm{Bin}(2N,;p_t).
    ]

    1) 先推 (\mathbb{E}[p_{t+1}\mid p_t])¶

    1.1 先推 (\mathbb{E}[K_{t+1}\mid p_t])（指示变量法）¶

    令第 (i) 次抽样的指示变量：

    [
    I_i=
    \begin{cases}
    1, & \text{第 } i \text{ 次抽到 } A\
    0, & \text{否则}
    \end{cases}
    \qquad i=1,\dots,2N
    ]

    那么

    [
    K_{t+1}=\sum_{i=1}^{2N} I_i.
    ]

    在给定 (p_t) 的条件下，
    [
    \mathbb{P}(I_i=1\mid p_t)=p_t,\quad \mathbb{P}(I_i=0\mid p_t)=1-p_t,
    ]
    所以

    [
    \mathbb{E}[I_i\mid p_t]=1\cdot p_t+0\cdot(1-p_t)=p_t.
    ]

    用期望线性性（这一步不需要独立性）：

    [
    \mathbb{E}[K_{t+1}\mid p_t]
    =\mathbb{E}!\left[\sum_{i=1}^{2N} I_i ,\middle|, p_t\right]
    =\sum_{i=1}^{2N}\mathbb{E}[I_i\mid p_t]
    =\sum_{i=1}^{2N} p_t
    =2N p_t.
    ]

    1.2 再得到 (\mathbb{E}[p_{t+1}\mid p_t])¶

    [
    p_{t+1}=\frac{K_{t+1}}{2N}
    \Rightarrow
    \mathbb{E}[p_{t+1}\mid p_t]
    =\mathbb{E}!\left[\frac{K_{t+1}}{2N}\middle|p_t\right]
    =\frac{1}{2N}\mathbb{E}[K_{t+1}\mid p_t]
    =\frac{1}{2N}\cdot 2N p_t
    =p_t.
    ]

    2) 推 (\mathbb{E}[p_{t+1}-p_t\mid p_t]=0)¶

    这一步其实是上面结果的直接推论，因为 (p_t) 在条件里是常数：

    [
    \mathbb{E}[p_{t+1}-p_t\mid p_t]
    =\mathbb{E}[p_{t+1}\mid p_t]-p_t
    =p_t-p_t
    =0.
    ]

    直觉：纯漂变不会在平均意义上推动频率上升或下降；它只带来随机波动。

    3) 推 (\mathrm{Var}(p_{t+1}-p_t\mid p_t)=\dfrac{p_t(1-p_t)}{2N})¶

    我们分两步：先算 (\mathrm{Var}(p_{t+1}\mid p_t))，再变成增量的方差。

    3.1 先算 (\mathrm{Var}(K_{t+1}\mid p_t))¶

    仍用指示变量法，并用到 WF 的独立抽样假设（这一步需要独立性来让协方差为 0）：

    [
    K_{t+1}=\sum_{i=1}^{2N} I_i.
    ]

    给定 (p_t)，(I_i\sim \mathrm{Bernoulli}(p_t))，所以

    [
    \mathrm{Var}(I_i\mid p_t)=p_t(1-p_t).
    ]

    并且在标准 WF 中 (I_i) 相互独立，因此 (i\neq j) 时
    [
    \mathrm{Cov}(I_i,I_j\mid p_t)=0.
    ]

    于是

    [
    \mathrm{Var}(K_{t+1}\mid p_t)
    =\mathrm{Var}!\left(\sum_{i=1}^{2N} I_i \middle| p_t\right)
    =\sum_{i=1}^{2N}\mathrm{Var}(I_i\mid p_t)
    +2\sum_{i<j}\mathrm{Cov}(I_i,I_j\mid p_t)
    ]

    协方差项为 0，得到

    [
    \mathrm{Var}(K_{t+1}\mid p_t)
    =2N\cdot p_t(1-p_t).
    ]

    （这也等价于二项分布 (\mathrm{Bin}(2N,p_t)) 的方差公式。）

    3.2 再算 (\mathrm{Var}(p_{t+1}\mid p_t))¶

    因为 (p_{t+1}=K_{t+1}/(2N))，用缩放性质 (\mathrm{Var}(aX)=a^2\mathrm{Var}(X))：

    [
    \mathrm{Var}(p_{t+1}\mid p_t)
    =\mathrm{Var}!\left(\frac{K_{t+1}}{2N}\middle|p_t\right)
    =\frac{1}{(2N)^2}\mathrm{Var}(K_{t+1}\mid p_t)
    =\frac{1}{(2N)^2}\cdot 2N p_t(1-p_t)
    =\frac{p_t(1-p_t)}{2N}.
    ]

    3.3 最后变成增量的方差¶

    注意：在条件 (p_t) 下，(p_t) 自身是常数，所以

    [
    \mathrm{Var}(p_{t+1}-p_t\mid p_t)
    =\mathrm{Var}(p_{t+1}\mid p_t)
    ]

    理由是：对任何常数 (c)，(\mathrm{Var}(X-c)=\mathrm{Var}(X))。

    所以直接得到

    [
    \mathrm{Var}(p_{t+1}-p_t\mid p_t)=\frac{p_t(1-p_t)}{2N}.
    ]

    4) 你现在得到的两条在 diffusion 里怎么用（先给你一个定位）¶

    把 (\Delta p_t:=p_{t+1}-p_t)，那么：

    • (\mathbb{E}[\Delta p_t\mid p_t]=0) → 漂移项（drift term）为 0（中性）

    • (\mathrm{Var}(\Delta p_t\mid p_t)=\dfrac{p_t(1-p_t)}{2N}) → 扩散强度随 (p(1-p)) 变（边界处为 0）

    下面我用严格的量级（big-O）公式把两件事讲清楚：

    1. 为什么 (N) 大时 WF 单代漂变弱；

    2. 为什么要把时间缩放成 (\tau=t/(2N))。

    1) 为什么 (N) 很大时，WF 的单代漂变“很弱”？¶

    在中性 WF 下（给定 (p_t)）我们已经有：

    [
    \mathbb{E}[\Delta p_t\mid p_t]=0,\qquad
    \mathrm{Var}(\Delta p_t\mid p_t)=\frac{p_t(1-p_t)}{2N},
    ]
    其中 (\Delta p_t:=p_{t+1}-p_t)。

    1.1 用方差量级说明“每代波动很小”¶

    因为 (0\le p_t(1-p_t)\le \tfrac14)，所以

    [
    \mathrm{Var}(\Delta p_t\mid p_t)\le \frac{1}{8N}=O!\left(\frac{1}{N}\right).
    ]

    更直观地看“典型步长”（标准差）：

    [
    \mathrm{SD}(\Delta p_t\mid p_t)=\sqrt{\mathrm{Var}(\Delta p_t\mid p_t)}
    =\sqrt{\frac{p_t(1-p_t)}{2N}}
    =O!\left(\frac{1}{\sqrt{N}}\right).
    ]

    这句话就是：单代频率变化的典型规模是 (1/\sqrt{N})，所以 (N) 越大，每代变化越小，漂变越“弱”。

    你要的 (O(1/N)) 也出现了：是方差（二阶矩）量级；而“幅度”是方差开根号，量级是 (O(1/\sqrt{N}))。

    2) 为什么要用 (\tau=t/(2N)) 这种尺度，而不是直接用 (t)？¶

    核心理由：如果不缩放时间，(N\to\infty) 时过程会退化成“几乎不动”；缩放后才能得到非平凡极限（扩散过程）。

    2.1 看累计波动：固定 (t) 时，(N\to\infty) 变化会消失¶

    考虑 (t) 代后的累计变化
    [
    p_t-p_0=\sum_{k=0}^{t-1}\Delta p_k.
    ]

    粗略但关键的量级计算：每一代 (\Delta p_k) 的方差是 (O(1/N))，那么 (t) 代累计方差大约是

    [
    \mathrm{Var}(p_t-p_0)\approx \sum_{k=0}^{t-1} O!\left(\frac{1}{N}\right)
    = O!\left(\frac{t}{N}\right).
    ]

    如果你把 (t) 当成“常数”（不随 (N) 增长），那

    [
    \mathrm{Var}(p_t-p_0)=O!\left(\frac{1}{N}\right)\xrightarrow[N\to\infty]{}0.
    ]

    也就是：在固定代数尺度上，群体越大，频率轨迹越接近常数（漂变被“压扁”了）。这就是为什么不能直接用 (t) 来取极限：极限会变得无聊。

    2.2 让极限“有东西发生”：需要让 (t) 与 (N) 同阶¶

    要让累计波动保持 (O(1))（既不爆炸也不消失），我们希望

    [
    \mathrm{Var}(p_t-p_0)\sim O(1).
    ]

    由上面的量级 (O(t/N))，这要求

    [
    \frac{t}{N}=O(1)\quad \Longrightarrow\quad t=O(N).
    ]

    WF 里自然出现的常数是 (2N)（因为二倍体有 (2N) 拷贝，并且方差里就是 (2N)），所以我们令

    [
    t = 2N\tau \quad\Longleftrightarrow\quad \tau=\frac{t}{2N}.
    ]

    把这个代回累计方差的量级：

    [
    \mathrm{Var}(p_{2N\tau}-p_0)\approx O!\left(\frac{2N\tau}{N}\right)=O(\tau),
    ]

    这就完美了：在 (\tau) 尺度上，波动不会因为 (N\to\infty) 而消失，于是才可能收敛到一个非退化的 diffusion。

    2.3 再用“每单位 (\tau) 的方差增量”解释为什么是 (2N)¶

    1 代对应的 (\tau) 增量是
    [
    \Delta\tau = \frac{1}{2N}.
    ]

    我们已经有单代
    [
    \mathrm{Var}(\Delta p_t\mid p_t)=\frac{p_t(1-p_t)}{2N}.
    ]

    于是“每单位 (\tau)”的方差增长率（把方差除以时间步长）是

    [¶

    \frac{\mathrm{Var}(\Delta p_t\mid p_t)}{\Delta\tau}

    \frac{\frac{p_t(1-p_t)}{2N}}{\frac{1}{2N}}
    = p_t(1-p_t),
    ]

    这是一个 (O(1)) 的量，不依赖 (N)。这正是你希望在极限里留下来的扩散系数。

    如果你不用 (\tau=t/(2N))，而用 (\tau=t/N) 或 (\tau=t)，这个“方差增长率”要么多一个常数因子（还好但不标准），要么趋于 0（退化），所以传统上用 (2N) 来把系数写得最干净。

    你可以记成两句“硬逻辑链”¶

    • 单代漂变的二阶矩：
      [
      \mathrm{Var}(\Delta p\mid p)=\frac{p(1-p)}{2N}=O!\left(\frac{1}{N}\right)
      \Rightarrow \Delta p = O!\left(\frac{1}{\sqrt{N}}\right).
      ]

    • (t) 代累计波动：
      [
      \mathrm{Var}(p_t-p_0)\approx O!\left(\frac{t}{N}\right)
      \Rightarrow t \text{ 必须取 } O(N) \text{ 才有非平凡极限}
      \Rightarrow \tau=\frac{t}{2N}.
      ]

    如果你接下来要把它推进到 diffusion，我可以直接把这套量级推导变成标准形式：
    用 (\tau=t/(2N)) 定义插值过程 (p^{(N)}(\tau))，然后写出它的 infinitesimal mean/variance（生成元），最后得到 Fokker–Planck 或 SDE。你想先看 生成元（generator） 的推法，还是直接从 SDE 写起？
